ACL Permission List
===================

On Anything
-----------
anything            can do anything
read                can access fundamental properties of an item
write               can change fundamental properties of an item

destroy             can erase an item
entrust             can change the ownership of an item
grant               can set permissions for an item

On Objects:
-------------
move                can change the location of an object
transmute           can change the parents of an object
derive              can change the children of an object

On Verbs:
----------
execute             can run this verb
develop             can modify this verb
