def isLocal(): (source)
Used by get/setattr to delegate access.

Returns True if the call to __getattribute__ or __setattr__ originated in either the exchange, test, or bootstrap modules.

Previously this used inspect, which made it super slow. The only potential issue with using _getframe() is that it's not guaranteed to be there in non CPython implementations.
