def connect(*args, db_urls=None, async=False, **kwargs): (source)
Get a new connection pool for a particular db_url.

Accepts a DSN specified as a URL, and returns a new connection pool object. If `async` is False (the default), all database requests are made synchronously.

Parametersdb_urlA URL of the form drivername://user:password@host/dbname (type: str )
asyncif True, create an asynchronous connection pool object. (type: bool )
*argsother positional arguments to pass to the DB-API driver.
**kwargsother keyword arguments to pass to the DB-API driver.
