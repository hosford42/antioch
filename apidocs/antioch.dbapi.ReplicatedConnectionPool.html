a.d.ReplicatedConnectionPool(object) : class documentation

Part of antioch.dbapi View Source View In Hierarchy

A collection of database servers in a replicated environment.

Queries are examined to see if there are SELECTs, with all reads being sent to the slaves in round-robin mode, and all writes are sent to the master.

When write_only_master is True, the master server is not used for reads. Otherwise it is included in the round-robin of servers to read from.

If there are no slaves, all queries are sent to the master.
