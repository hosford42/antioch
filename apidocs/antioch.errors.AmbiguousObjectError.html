a.e.AmbiguousObjectError(UserError) : class documentation

Part of antioch.errors View In Hierarchy

When this class is raised, it means that at some point a single object was expected, but multiple ones were found instead. When printed to the user, this shows a list of matching items, along with their unique IDs, so the user can choose the correct one.
