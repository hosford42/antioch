a.c.ClientInterface(athena.LivePage) : class documentation

Part of antioch.client View Source View In Hierarchy

The primary client interface.

This page maintains an Athena connection to the server by using the LiveElement `ClientConnector`. If this page is closed, any child windows will stop working, as they use the parent's connection to the server, instead of having their own.
